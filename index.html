Two Way Data Binding

Tradtionally when the model changes, the developer is responsible for updating DOM elements to reflect these changes.

This is also a two way street, when the DOM is changed, we need to update the model to reflect the changes.

AngularJS handles synchronization between the DOM and model and vice versa.
